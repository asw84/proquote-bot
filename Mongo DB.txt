Шаг 2: Первоначальное заполнение MongoDB
Перед первым запуском необходимо вручную создать записи для пользователей и конфигурации в вашей базе данных MongoDB Atlas.
Зайдите в "Browse Collections".
Создайте базу данных (например, proquote_db) и в ней коллекцию users.
В коллекцию users добавьте документ для администратора:

{
  "user_id": ВАШ_TELEGRAM_ID,
  "is_active": true,
  "name": "Admin User"
}
